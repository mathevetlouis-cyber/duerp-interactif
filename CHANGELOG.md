# Changelog

Toutes les modifications notables de ce projet sont documentées dans ce fichier.

Le format est basé sur [Keep a Changelog](https://keepachangelog.com/fr/1.0.0/).

## [2.0] - 2025

### Ajouté
- Questionnaire RPS complet (20 questions) selon les 6 dimensions du rapport Gollac (2011)
- Double matrice de criticité (risques classiques + RPS)
- Synthèse RPS dédiée avec statistiques
- Onglets de navigation Risques / RPS par agent
- Système de collapsibles pour les catégories de questions
- Plan d'actions intégré avec filtrage par type (risques / RPS)
- Export Word du plan d'actions
- Indicateur temps réel
- Navigation par agent avec badges de complétion

### Modifié
- Architecture complète refondée en application monopage à onglets
- 14 unités de travail prédéfinies pour les collectivités territoriales
- Interface responsive améliorée

## [1.0] - 2024

### Ajouté
- Questionnaire risques professionnels (30 questions, 12 familles)
- Matrice de criticité Fréquence × Gravité
- Évaluation multi-agents avec agrégation
- Export JSON (sauvegarde/chargement)
- Export PDF (impression)
- Stockage local (localStorage)
