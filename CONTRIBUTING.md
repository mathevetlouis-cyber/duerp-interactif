# Contribuer au projet DUERP Interactif

Merci de votre intérêt pour ce projet.

## Signaler un problème

Utilisez les [Issues GitHub](../../issues) pour signaler un bug ou proposer une amélioration.

## Proposer une modification

1. Forkez le dépôt
2. Créez une branche (`git checkout -b feature/amelioration`)
3. Committez vos modifications (`git commit -am 'Ajout fonctionnalité X'`)
4. Poussez vers la branche (`git push origin feature/amelioration`)
5. Ouvrez une Pull Request

## Règles

- L'application doit rester un fichier HTML unique, sans dépendance externe
- Les textes doivent être en français
- Toute modification doit être testée sur les navigateurs principaux (Chrome, Firefox, Safari, Edge)
- Les données utilisateur ne doivent jamais quitter le navigateur (localStorage uniquement)
